This section covers every versioned release of the Nexa Ed SDK packages. Each release page documents what changed, what was added, and any breaking changes or migration steps you need to know about.

SDK packages follow [Semantic Versioning](https://semver.org/). Until `1.0.0` is reached, minor version bumps (`0.x.0`) may introduce API changes — check the migration notes on each release page before upgrading.
