# create-nexaed-app

Scaffold a Nexa-connected school app in seconds.

## Usage

```bash
npx create-nexaed-app@latest
# or
pnpm create nexaed-app
```

Follow the interactive prompts to choose your framework and generate a ready-to-run project pre-wired to the [Nexa Education Platform](https://docs.nexa-ed.com).
